## KYC 身分認證：Lv.1 與 Lv.2 的差異與操作

KYC（Know Your Customer）是金融機構與交易所依法必須做的身分查核，目的在防制洗錢。OKX 把 KYC 分成兩級，等級越高，能用的功能與提幣額度越高。

### KYC Lv.1：基礎身分認證

進入 App 左上角頭像 → 身分認證 → Lv.1「去認證」。系統會要求你輸入真實姓名、國家、證件號碼等基本資料，通常幾分鐘內就會審核通過。Lv.1 完成後可解鎖基本交易與出入金功能。

### KYC Lv.2：進階身分認證（含人臉辨識）

Lv.1 通過後會看到「開始 Lv.2 認證」。你需要：

1. 上傳證件正面與背面照片（建議光線充足、四角入鏡）

2. 用手機或電腦鏡頭做人臉辨識，確認你與證件上照片為同一人

Lv.2 審核時間通常幾小時內完成，通過後可享有更高的提幣限額與完整交易權限。

> 小提醒：證件照模糊、人臉辨識角度偏差、資料與證件對不上，是審核失敗最常見的原因。失敗了別緊張，重新拍照上傳即可，也可找客服協助。

## 現貨交易入門：第一筆買賣這樣做

完成入金後，資金會先進到「資金帳戶」。要進行現貨交易，需先把幣劃轉到「交易帳戶」。

**劃轉步驟**：App 首頁 → 資產 → 資金劃轉 → 選擇幣種與數量 → 確認。

**現貨買賣步驟**：

1. App 底部連點兩次「交易」→ 選「現貨」

2. 搜尋交易對（例如 BTC/USDT）

3. 選擇買入方式：

- **限價單（Maker）**：自己設價格，掛單等成交，手續費較低

- **市價單（Taker）**：以當下市場價格直接成交，手續費較高

4. 輸入數量或金額 → 點買入 / 賣出

> 新手建議先用小額練習限價單，熟悉介面與成交邏輯後再放大資金。

## 合約交易新手須知

合約是 OKX 衍生品市場的主力，分永續合約與交割合約，又有 U 本位（USDT 結算）與幣本位（以幣結算）兩種。合約可開槓桿放大收益，但風險也同步放大，新手務必先理解再下手。

合約交易基本流程：

1. 資金劃轉到「交易帳戶」

2. 進入合約頁，選擇合約類型（建議新手從 U 本位永續合約開始）

3. 設定槓桿倍數（新手建議 2~3 倍，不要一上來就 10 倍以上）

4. 設定保證金模式（逐倉或全倉）

5. 輸入開倉數量與價格 → 點買漲（做多）或買跌（做空）

> 強烈建議：合約是高風險工具，第一次操作前先用 OKX 的「模擬交易」功能練手，熟悉爆倉、保證金、強平邏輯後再投入真實資金。

## 跟單交易：讓專業交易員幫你操作

如果合約自己不敢玩，OKX 有「跟單交易」功能，可以一鍵複製專業交易員的策略。OKX 跟單分一般跟單與智能跟單兩種模式，智能跟單只需設定投資額，槓桿與倉位由交易員自動同步管理。

挑選交易員時建議看幾個指標：

- 收益率與收益額

- 勝率

- 帶單規模與當前跟單人數

- 最大回撤（衡量風險控制能力）

> 跟單不是保證獲利。交易員有順風期也有逆風期，績效數據可能被美化，跟單前請務必做好風險評估，並設定可承受的虧損上限。
